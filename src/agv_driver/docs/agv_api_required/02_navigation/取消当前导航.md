**取消当前导航**

**请求**

- 编号: 3003 (0x0BBB)
- 名称: robot\_task\_cancel\_req
- 描述: 取消当前导航
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 0B BB 00 00 00 00 00 00```

**响应**

- 编号: 13003 (0x32CB)
- 名称: robot\_task\_cancel\_res
- 描述: 取消当前导航的响应
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```JSON
5A 01 00 01 00 00 00 0E 32 CB 0B BB 00 00 00 00
// {"ret\_code":0}
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
