**配置机器人推送端口**

**请求**

- 编号: 9300
- 名称: robot\_push\_config\_req
- 描述: 配置推送端口 19301
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|interval|Interger|消息推送时间间隔（ms）|是|
|included\_fields|array[string]|设置消息中包含的字段|是|
|excluded\_fields|array[string]|设置消息中排除的字段|是|


```<p>警告：</p><p>included\_fields 和 excluded\_fields 不可同时设置</p>```

**请求示例**


```JSON
// 1）设置发布时间间隔为 2000 ms
{"interval":2000}

// 2）设置 include\_fields
{"included\_fields":["vx","vy"]}

// 3) 设置 excluded\_fields
{"excluded\_fields":["blocked"]}

// 4) 同时设置发布时间间隔 和 include\_fields
{"interval":2000,"included\_fields":["vx","vy"]}```

**响应**

- 编号: 19300
- 名称: robot\_push\_config\_res
- 描述: 配置推送端口的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```
