**查询当前控制权所有者**

**请求**

- 编号: 1060 (0x0424)
- 名称: robot\_status\_current\_lock\_req
- 描述: 查询当前控制权所有者
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 04 24 00 00 00 00 00 00```

**响应**

- 编号: 11060 (0x2B34)
- 名称: robot\_status\_current\_lock\_res
- 描述: 查询当前控制权所有者的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|locked|boolean|当前控制权是否被抢占|否|
|ip|string|控制权所有者 ip|是|
|port|Interger|控制权所有者端口号|是|
|type|uint8\_t|控制权所有者类型：<br>0x00=default<br>0x02=roboshop<br>0xDD=srd|是|
|nick\_name|string|控制权所有者昵称信息|是|
|time\_t|number|抢占控制权的时间戳(s)|是|
|desc|string|控制权所有者描述|是|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "desc": "",
  "ip": "183.192.159.190",
  "locked": true,
  "nick\_name": "srd-seer-mizhan",
  "port": 52681,
  "ret\_code": 0,
  "time\_t": 1611562267,
  "type": 0
}```


```JSON
{
  "locked": false
}```
