**软急停**


```<p>注意：</p><p>该 API 用于通过软件的方式控制机器人控制器是否输出急停信号。</p>```

**请求**

- 编号: 6004 (0x1774)
- 名称: robot\_other\_softemc\_req
- 描述: 软急停
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|status|boolean|true 为输出急停, false 为不输出急停|否|

**请求示例**


```JSON
5A 01 00 01 00 00 00 0F 17 74 00 00 00 00 00 00
{"status":true}
// 序列化后十六进制为：
7B 22 73 74 61 74 75 73 22 3A 74 72 75 65 7D```

**响应**

- 编号: 16004 (0x3E84)
- 名称: robot\_other\_softemc\_res
- 描述: 软急停的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 3E 84 17 74 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
