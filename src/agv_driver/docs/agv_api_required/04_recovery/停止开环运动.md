**停止开环运动**

**请求**

- 编号: 2000 (0x07D0)
- 名称: robot\_control\_stop\_req
- 描述: 停止开环运动
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 07 D0 00 00 00 00 00 00```

**响应**

- 编号: 12000 (0x2EE0)
- 名称: robot\_control\_stop\_res
- 描述: 停止开环运动的响应
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```Plaintext
5A 01 00 01 00 00 00 0E 2E E0 07 D0 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
