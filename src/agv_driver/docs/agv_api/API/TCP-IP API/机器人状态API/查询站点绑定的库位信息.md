**查询站点绑定的库位信息**

**请求**

- 编号: 1310 (0x051e)
- 名称: robot\_status\_station\_bin\_locations\_req
- 描述: 查询站点绑定的库位信息
- JSON 数据区: 见下表
- rbk版本：3.4.7.1002+

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|stations|array[string]|站点列表|否|

**请求示例**

```JSON
//{"stations":["LM1"]}
5A 01 00 00 00 00 00 14 05 1E 05 1E 00 00 00 00
7B 22 73 74 61 74 69 6F 6E 73 22 3A 5B 22 4C 4D 
31 22 5D 7D  ```
