**查询机器人信息**

**请求**

- 编号: 1000 (0x03E8)
- 名称: robot\_status\_info\_req
- 描述: 查询机器人信息
- JSON 数据区: 无

**请求示例**

数据区长度为 0, 只有包头:

```Plaintext
5A 01 00 01 00 00 00 00 03 E8 00 00 00 00 00 00 ```

**响应**

- 编号: 11000 (0x2AF8)
- 名称: robot\_status\_info\_res
- 描述: 查询机器人信息的响应
- JSON 数据区: 见下表

```如果响应数据区的 JSON Object 中有 API 中未提到的 key-value 键值对，可以直接忽略，无须关注其含义，下文不再赘述```
