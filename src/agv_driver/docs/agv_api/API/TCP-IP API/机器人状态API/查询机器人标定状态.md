**查询机器人标定状态**

**请求**

- 编号: 1510
- 名称: ROBOT\_STATUS\_CALIB\_STATUS\_REQ
- 描述: 查询机器人标定状态
- JSON 数据区: 无

**响应**

- 编号: 11510
- 名称: ROBOT\_STATUS\_CALIB\_STATUS\_RES
- 描述: 查询机器人标定状态的响应
- JSON 数据区: 见下

```JSON
{
    "calib\_status":{
        "status":1,  //0:无 1:正在标定 2:标定成功 3:标定失败
        "desc":"laser1"
    }
}```
