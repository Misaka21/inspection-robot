**查询编码器脉冲值**

**请求**

- 编号: 1018 (0x03FA)
- 名称: robot\_status\_encoder\_req
- 描述: 查询编码器脉冲值
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 03 FA 00 00 00 00 00 00```

**响应**

- 编号: 11018 (0x2B0A)
- 名称: robot\_status\_encoder\_res
- 描述: 查询编码器脉冲值的响应
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|~~encoder~~|~~array[int]~~|~~查询到的编码器脉冲值~~ 已弃用，改为motor\_encoder|是|
|motor\_encoder|array[object]|查询到的电机编码器，包含电机名和编码器脉冲值|是|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "encoder": [],
  "motor\_encoder": [],
  "ret\_code": 0
}```
