**查询机器人脚本列表**

**请求**

- 编号: 1506 (0x05E2)
- 名称: robot\_status\_script\_info\_req
- 描述: 查询机器人脚本列表
- JSON 数据区: 见下表

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|<p>type</p><p></p>|string|机器人脚本的类型，“module”=机构脚本，“ genetic ”=通用脚本|否|
**请求示例**

如下表示查询机器人机构脚本列表

```JSON
{
  "type": "module"
}```

**响应**

- 编号: 11506 (0x2CF2)
- 名称: robot\_status\_script\_info\_res
- 描述: 查询机器人脚本列表 的响应
- JSON 数据区: 见下

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|list|object array|为脚本列表的json数组，每一个脚本信息为一个json对象。其中包含脚本 md5:string ，脚本相对路径 relative\_path:string ，脚本类型 type:string 详情见示例。|是|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "create\_on": "2022-11-07T13:24:42.222Z",
  "list": [{
    "md5": "c37f5c383ecd80874e9e4a5431d93849",
    // 脚本md5
    "relative\_path": "crc.py",
    // 脚本相对路径
    "type": "module" // 脚本类型
  },
  {
    "md5": "d3898d6848a49eba38ac1f65b8cdbb37",
    "relative\_path": "ctuNoBlock.py",
    "type": "module"
  },
  {
    "md5": "d1537d9d1a1897b760814b16d7a5c192",
    "relative\_path": "params/ctuNoBlock.json",
    "type": "module"
  }],
  "ret\_code": 0
}```
