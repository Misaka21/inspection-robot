**查询批量数据1**

**请求**

- 编号: 1100 (0x044C)
- 名称: robot\_status\_all1\_req
- 描述: 查询批量数据1
- JSON 数据区: 见下表

all1 是为了通过一个请求批量获取之前提到的大部分状态数据, 包括 1000, 1001, 1002, 1003, 1004, 1005, 1006, 1007, 1008, 1009, 1010, 1011, 1012, 1013, 1014, 1015, 1016, 1017, 1019, 1020, 1021, 1022, 1025, 1027, 1028, 1029, 1030, 1050, 1051等

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|<p>keys</p><p></p>|array[string]|只返回数组中指定的 key-value，缺省则为返回所有数据|是|
|<p>return\_laser</p><p></p>|boolean|是否返回激光数据（即 1009 的数据），true = 返回，false = 不返回，缺省则为返回|是|
|<p>return\_beams3D</p><p></p>|boolean|是否返回多线激光数据（即 1009 中多线的数据），true = 返回，false = 不返回，缺省则为不返回|是|
|<p>return\_mid360</p><p></p>|<p>boolean</p><p></p>|是否返回mid360数据，true = 返回，false = 不返回，缺省则为不返回（3.4.6.1802及以后删除）|<p>是</p><p></p>|
|<p>timeout</p><p></p>|<p>int</p><p></p>|表示距离下次请求的超时时间，下一次请求会把上次请求的 "timeout" 时间重置，超时未进行查询后，报个 warning（55011）|<p>是</p><p></p>|
|language|string|||
