**查询机器人参数**

**请求**

- 编号: 1400 (0x0578)
- 名称: robot\_status\_params\_req
- 描述: 查询机器人参数信息
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|plugin|string|参数所属的插件名, 如果缺省, 表示查询所有插件的所有参数|是|
|param|string/array|参数名, 如果 plugin 存在，但 param 缺省，代表查询该插件的所有参数。否则查询该插件的指定参数。可以为单独的参数或者参数数组。|<p>是</p><p></p>|

**数据示例**

如下表示查询 MoveFactory 插件的 MaxAcc 参数

```JSON
{
    "plugin": "MoveFactory",
    "param": "MaxAcc"
}```


```JSON
{
    "param":{
        "MoveFactory":[

        ],
        "ArmAdapter":[
            "ArmOperationTimeout",
            "ArminfoPublishTime"
        ]
    }
}```


```JSON
{
    "param":{
        "MoveFactory":"", //返回 MoveFactory 全部数据
        "ArmAdapter":[   //返回 ArmAdapter 的ArmOperationTimeout 和 ArminfoPublishTime 数据
            "ArmOperationTimeout",
            "ArminfoPublishTime"
        ]
    }
}```

**响应**

- 编号: 11400 (0x2C88)
- 名称: robot\_status\_params\_res
- 描述: 查询机器人参数信息的响应
- JSON 数据区: 见下
