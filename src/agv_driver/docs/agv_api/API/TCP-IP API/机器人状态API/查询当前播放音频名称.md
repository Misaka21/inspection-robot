**查询当前播放音频名称**

**请求**

- 编号: 1850 (0x073A)
- 名称: robot\_status\_sound\_req
- 描述: 查询当前播放音频名称
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 00 00 00 00 00 07 3A 07 3A 00 00 00 00```

**响应**

- 编号: 11850 (0x2E4A)
- 名称: robot\_status\_sound\_res
- 描述: 查询当前播放音频名称的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|status|number|当前音频播放状态|是|
|sound\_name|string|当前播放音频的名称|是|
|loop|bool|音频播放是否循环|是|
|count|number|音频播放剩余次数|是|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "create\_on": "2022-06-10T14:45:41.849Z",
  "ret\_code": 0,
  "status": 0,
  "sound\_name": "",
  "loop": "false",
  "count": 0
}```
