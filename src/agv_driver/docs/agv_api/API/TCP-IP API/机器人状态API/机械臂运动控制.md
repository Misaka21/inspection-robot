**机械臂运动控制**

**请求**

- 编号: 1673 (0x0689)
- 名称: robot\_status\_armmove\_req
- 描述: 机械臂运动控制
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|type|string|<p>机械臂控制类型：</p><p>stop=停止</p><p>pause=暂停</p><p>resume=继续</p><p>move\_to\_saved\_point=回到保存位姿</p><p>J1+ = 控制手臂基座正向运动</p><p>J1- = 控制手臂基座负向运动</p><p>J2+ = 控制手臂肩部正向运动</p><p>J2- = 控制手臂肩部负向运动</p><p>J3+ = 控制手臂肘部正向运动</p><p>J3- = 控制手臂肘部负向运动</p><p>J4+ = 控制手臂手腕1正向运动</p><p>J4- = 控制手臂手腕1负向运动</p><p>J5+ = 控制手臂手腕2正向运动</p><p>J5- = 控制手臂手腕2负向运动</p><p>J6+ = 控制手臂手腕3正向运动</p><p>J6- = 控制手臂手腕3负向运动</p><p>X+ = 控制手臂沿着末端X轴正向运动</p><p>X- = 控制手臂沿着末端X轴负向运动</p><p>Y+ = 控制手臂沿着末端Y轴正向运动</p><p>Y- = 控制手臂沿着末端Y轴负向运动</p><p>Z+ = 控制手臂沿着末端Z轴正向运动</p><p>Z- = 控制手臂沿着末端Z轴负向运动</p><p>RX+ = 控制手臂沿着末端X轴顺时针旋转</p><p>RX- = 控制手臂沿着末端X轴逆时针旋转</p><p>RY+ = 控制手臂沿着末端Y轴顺时针旋转</p><p>RY- = 控制手臂沿着末端Y轴逆时针旋转</p><p>RZ+ = 控制手臂沿着末端Z轴顺时针旋转</p><p>RZ- = 控制手臂沿着末端Z轴逆时针旋转</p><p>Jog\_Stop = 取消控制机械臂移动</p><p>set\_speed\_slider = 设置机械臂运行速率滑块</p><p></p>|否|
|value|double|机械臂的运动快慢程度，仅支持[0.02, 1]，double类型。只有type为set\_speed\_slider才生效。||
**请求示例**

暂停机械臂

```JSON
{
    "type": "pause"
}```
设置机械臂运动速率

```JSON
{
    "value":0.4,
    "type":"set\_speed\_slider"
}```
**响应**

- 编号: 11673 (0x2D99)
- 名称: robot\_status\_armmove\_res
- 描述: 机械臂运动控制的响应
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|
