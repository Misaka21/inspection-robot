**查询机器人脚本详情列表**

**请求**

- 编号: 1507 (0x05E3)
- 名称: robot\_status\_script\_detailslist\_req
- 描述: 查询机器人脚本详情列表
- JSON 数据区: 见下表

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|<p>type</p><p></p>|<p>string</p><p></p>|机器人脚本的类型，“module”=机构脚本，“ genetic ”=通用脚本|否|


```目前响应中的 default\_args 和 description 只对机构脚本有效```

**请求示例**

如下表示查询机器人机构脚本详情列表

```JSON
{
        "type": "module"
}```

**响应**

- 编号: 11507 (0x2CF3)
- 名称: robot\_status\_script\_detailslist\_res
- 描述: 查询机器人脚本详情列表 的响应
- JSON 数据区: 见下

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|list|object array|为脚本列表的json数组，每一个脚本信息为一个json对象。其中包含脚本 md5:string ，脚本相对路径 name:string ，脚本默认参数 default\_args:json ，脚本说明 description:string 详情见示例。|是|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|


```需注意，若参数说明和脚本说明中包含中文，则响应中会将中文转换成unicode编码字符串。需要用户手动转换成原文。```

**响应示例**


```JSON
{
  "create\_on": "2022-11-07T13:47:08.338Z",
  "list": [{
    "default\_args": {
      "move\_task\_list": {
        "tips": "\u4efb\u52a1",
        // unicode：任务
        "type": "json",
        "unit": "",
        "value": ""
      }
    },
    "description": "this is the description for this script",
    "md5": "17e33f63b7dc51337b6d7d3a5ca10ab4",
    "name": "syspy/addMoveTaskList.py"
  }],
  "ret\_code": 0
}```
