**查询相机标定图片**

**请求**

- 编号: 1668 (0x0684)
- 名称: robot\_status\_cameradata\_req
- 描述: 查询相机标定图片
- JSON 数据区: 见下表

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|<p>type</p><p></p>|string|图片类型，目前支持"2D\_image","2D\_recognition","recognize\_all"|否|
|<p>tag</p><p></p>|string|识别文件，仅在 type 为 "2D\_recognition" 或 "recognize\_all" 时使用|否|
**请求示例**

```C++
//2D\_image
{
    "type":"2D\_image"
}

//recognize\_all
{
    "tag": "chargerspot/fork.chargerspot",
    "type": "recognize\_all"
}```

**响应**

- 编号: 11668 (0x2D94)
- 名称: robot\_status\_cameradata\_res
- 描述: 查询相机标定图片的响应
- JSON 数据区: 见下
- 如果不发生错误, 数据区中的内容为图片数据
- 请求失败时返回下表所示 json

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**

```C++
{
    "create\_on": "2025-09-04T09:25:11.154+0800",
    "err\_msg": "param: not supported yet",
    "ret\_code": 40003
}```
