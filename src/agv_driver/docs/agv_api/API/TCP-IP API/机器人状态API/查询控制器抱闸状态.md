**查询控制器抱闸状态**

**请求**

- 编号: 1008 (0x03F0)
- 名称: robot\_status\_barke\_req
- 描述: 查询控制器抱闸状态
- JSON 数据区:无

**请求示例**

```JSON
5A 01 00 00 00 00 00 00 03 F0 03 F0 00 00 00 00 ```

**响应**

- 编号: 11008(2B00)
- 名称: robot\_status\_barke\_res
- 描述: 查询控制器抱闸状态的响应
- JSON 数据区: 见下表

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|barke|bool|抱闸状态|否|
**响应示例**

```JSON
{
    "brake": false,
    "create\_on": "2025-09-26T11:49:15.048+0800",
    "ret\_code": 0
}```
