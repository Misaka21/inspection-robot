**仿真从文件识别**

**请求**

- 编号: 1676 (0x068C)
- 名称: robot\_status\_recofiles\_req
- 描述: 仿真从文件识别
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|type|string|<p>请求类型：</p><p>getRecoFilesName =获取文件列表</p><p>recoFromFile =从文件识别</p><p></p>|否|
|fileExtension|string|文件后缀，类型为 getRecoFilesName时使用，为pcd或jpg|否|
|filesName|array|文件名，类型为recoFromFile时使用|否|
|recoModelPath|string|模型路径，类型为recoFromFile时使用|否|
**请求示例**

获取文件列表

```JSON
{
        "type":  "getRecoFilesName",
        "fileExtension": "pcd"                //pcd或者jpg           
}```
从文件识别

```JSON
{
  "type":  "recoFromFile",
  "filesName": [
    "/usr/local/.Seer/roboview/1.pcd",
    "/usr/local/.Seer/roboview/2.pcd"
  ]
  "recoModelPath":"pallet/p0001.pallet"
}```
**响应**

- 编号: 11675 (0x2D9C)
- 名称: robot\_status\_recofiles\_res
- 描述: 仿真从文件识别的响应
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|type|string|<p>请求类型：</p><p>getRecoFilesName =获取文件列表</p><p>recoFromFile =从文件识别</p>|否|
|filesName|array|文件名，类型为 getRecoFilesName时使用|否|
|results|string|rbk::protocol::Message\_RecognizeResultList 格式序列化后的字符串，类型为recoFromFile时使用|否|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|
**响应示例**

获取文件列表

```JSON
{
  "type":  "getRecoFilesName",
  "filesName": [
    "/usr/local/.Seer/roboview/1.pcd",
    "/usr/local/.Seer/roboview/2.pcd"
  ]
}```
从文件获取

```JSON
{
  "type":  "recoFromFile",
  "results":                   //rbk::protocol::Message\_RecognizeResultList 格式序列化后的字符串        
}```
