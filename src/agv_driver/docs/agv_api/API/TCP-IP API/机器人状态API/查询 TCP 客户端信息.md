**查询 TCP 客户端信息**

**请求**

- 编号: 1001 (0x03E9)
- 名称: robot\_status\_client\_req
- 描述: 查询 TCP 客户端信息
- JSON 数据区:无

**请求示例**

```JSON
5A 01 00 00 00 00 00 D3 2A F9 03 E9 00 00 00 00```

**响应**

- 编号: 11001(2AF9)
- 名称: robot\_status\_client\_res
- 描述: 查询 TCP 客户端信息的响应
- JSON 数据区: 见下表

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|19204|number|当前 19204 端口连接的客户端数量|否|
|19205|number|当前 19205 端口连接的客户端数量|否|
|19206|number|当前 19206 端口连接的客户端数量|否|
|19207|number|当前 19207 端口连接的客户端数量|否|
|19210|number|当前 19210 端口连接的客户端数量|否|
|19204\_ip|string|当前 19204 端口连接的客户端|否|
|19205\_ip|string|当前 19205 端口连接的客户端|否|
|19206\_ip|string|当前 19206 端口连接的客户端|否|
|19207\_ip|string|当前 19207 端口连接的客户端|否|
|19210\_ip|string|当前 19210 端口连接的客户端|否|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|
