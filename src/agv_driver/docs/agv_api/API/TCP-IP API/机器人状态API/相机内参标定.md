**相机内参标定**

**请求**

- 编号: 1906 (0x0772)
- 名称: robot\_status\_get\_param\_for\_calib
- 描述: 相机内参标定
- JSON 数据区: 见下表

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|<p>type</p><p></p>|string|动作类型，根据识别定义|否|
**请求示例**

```JSON
{
  "type": "start\_img"
}```
**响应**

- 编号: 11906 (0x2E08)
- 名称: robot\_status\_get\_param\_for\_calib\_res
- 描述: 相机内参标定的响应
- JSON 数据区: 见下

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|


```<p>当 type 为 start\_img 或 start\_board\_msg 时，rbk 会订阅识别插件的 Message\_CalibProperty 消息，每获取到一帧消息，都会主动上传给请求的端口，直到收到 type 为 stop\_img 或 stop\_board\_msg 的请求。</p><p>主动上传消息编号为17003，内容为序列化后的 Message\_CalibProperty。</p>```
**响应示例**


```JSON
{
  "create\_on": "2022-11-07T13:24:42.222Z",
  "ret\_code": 0
}```
