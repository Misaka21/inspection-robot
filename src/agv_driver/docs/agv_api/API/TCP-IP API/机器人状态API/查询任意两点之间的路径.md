**查询任意两点之间的路径**

**请求**

- 编号: 1303 (0x0517)
- 名称: robot\_status\_get\_path\_req
- 描述: 查询任意两点之间的路径
- JSON 数据区: 见下表
- rbk版本：3.4.5.26+

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|target\_id|string|线路的终点|否|
|source\_id|string|线路的起点|否|

**请求示例**



```JSON
//{"source\_id": "LM1", "target\_id":  "LM10"}
5A 01 00 01 00 00 00 00 05 17 00 00 00 00 00 00 
7B 22 73 6F 75 72 63 65 5F 69 64 22 3A 20 22 4C 
4D 31 22 2C 20 22 74 61 72 67 65 74 5F 69 64 22 
3A 20 20 22 4C 4D 31 30 22 7D```

**响应**

- 编号: 11303 (2c27)
- 名称: robot\_status\_info\_res
- 描述: 查询任意两点之间的路径的响应
- JSON 数据区: 见下表

```如果响应数据区的 JSON Object 中有 API 中未提到的 key-value 键值对，可以直接忽略，无须关注其含义，下文不再赘述```


```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|list|array[string]|站点数组|是|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**

```JSON

{
  "create\_on":"2023-08-17T03:36:40.770-0700",
  "err\_msg":"param: target\_id or source\_id is null",
  "ret\_code":40002
}```
