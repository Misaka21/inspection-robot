**查询modbus数据**

**请求**

- 编号: 1071 (0x042F)
- 名称: ROBOT\_STATUS\_MODBUS\_REQ
- 描述: 查询modbus数据
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|4x|json|json中为需要查询的位数，暂时只处理可写寄存器（4x)|是|

**请求示例**


```Plaintext
{"4x":[121,122]}```
或

```Plaintext
{"4x":["121","122"]}```

**响应**

- 编号: 11071 (0x2B3F)
- 名称: ROBOT\_STATUS\_MODBUS\_REQ
- 描述: 查询modbus数据
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|4x|json|查询到的寄存器值|否|

**响应示例**


```JSON
{"4x":{"121":1,"122":2}}```
