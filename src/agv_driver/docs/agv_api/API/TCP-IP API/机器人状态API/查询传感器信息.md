**查询传感器信息**

**请求**

- 编号: 1105(0x042551)
- 名称: robot\_status\_sensors\_data\_req
- 描述:查询传感器信息
- JSON 数据区:见下表

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|sensors|json|传感器列表|否|
sensors 字段内容：

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|depth\_cameras|json array|深度相机列表|否|
**请求示例**

```JSON
5A 01 00 00 00 00 00 00 04 51 04 51 00 00 00 00
{
    "sensors": {
        "depth\_cameras": [
            "camera"
        ]
    }
}```

- 描述:查询传感器信息的响应
