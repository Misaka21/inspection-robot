**下载机器人文件**

**请求**

- 编号: 1800 (0x0708)
- 名称: robot\_status\_downloadfile\_req
- 描述: 下载机器人文件
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|type|string|机器人文件的类型，“app”=机器人程序目录下的文件，“users”=机器人用户目录下的文件|是|
|file\_path|string|机器人文件的相对路径，针对于 type 指定的文件类型|是|

**请求示例**

如下表示下载机器人程序目录下的翻译脚本文件 rbk.ts

```JSON
{
    "type": "app",
    "file\_path": "rbk.ts"
}```


```JSON
5A 01 00 00 00 00 00 23 07 08 00 00 00 00 00 00 
7B 22 74 79 70 65 22 3A 22 61 70 70 22 2C 22 66 69 6C 65 5F 70 61 74 68 22 3A 22 72 62 6B 2E 73 68 22 7D```

**响应**

- 编号: 11800 (0x2E18)
- 名称: robot\_status\_downloadfile\_res
- 描述: 下载机器人文件的响应
- JSON 数据区: 见下

如果不发生错误, 数据区中的内容为文件内容

若发生错误，则响应示例如下：

**响应示例**


```JSON
{
    "create\_on": "2021-11-10T15:45:30.077Z",
    "err\_msg": "file not exist",
    "ret\_code": 40055
}```
