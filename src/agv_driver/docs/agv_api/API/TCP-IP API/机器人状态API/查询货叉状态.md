**查询货叉状态**

**请求**

- 编号: 1031(0x0407)
- 名称: robot\_status\_hook\_req
- 描述:查询货叉状态
- JSON 数据区:无

**请求示例**

```JSON
5A 01 00 00 00 00 00 00 04 07 04 07 00 00 00 00```

**响应**

- 编号: 11031(0x2B17)
- 名称: robot\_status\_hook\_res
- 描述:查询货叉状态的响应
- JSON 数据区: 见下表

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|hook\_mode|bool|货叉模式|否|
|hook\_state|number|货叉|否|
|hook\_enable|bool|是否启用货叉|否|
|hook\_error\_code|number|货叉错误码|否|
|hook\_isFull|bool|货叉是否有料|否|
|hook\_angle|number|货叉角度|否|
|hook\_height|number|货叉高度|否|
|hook\_clamping\_state|bool|货叉加紧状态|否|
|hook\_emc|bool|货叉是否急停|否|
|peripheral\_data|json array|外设状态|否|
**响应示例**

```JSON
{
    "create\_on": "2025-09-26T14:44:00.755+0800",
    "hook\_angle": 0,
    "hook\_clamping\_state": false,
    "hook\_emc": false,
    "hook\_enable": false,
    "hook\_error\_code": 0,
    "hook\_height": 0,
    "hook\_isFull": false,
    "hook\_mode": false,
    "hook\_state": 0,
    "peripheral\_data": [
    ],
    "ret\_code": 0
}```
