**查询标定状态**

**请求**

- 编号: 1023(0x03FF)
- 名称: robot\_status\_calibration\_req
- 描述: 查询标定状态
- JSON 数据区:无

**请求示例**

```JSON
5A 01 00 00 00 00 00 00 03 FF 03 FF 00 00 00 00```

**响应**

- 编号: 11023(0x2B0F)
- 名称: robot\_status\_calibration\_res
- 描述:查询标定状态的响应
- JSON 数据区: 见下表

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|calib\_status|json|旧版本标定状态|否|
|calib2\_status|json|新版本标定状态（如有此字段则忽略calib\_status)|是|
**响应示例**

```JSON
{
    "calib2\_status": {
        "desc": "",
        "status": 0
    },
    "calib\_status": {
        "desc": "",
        "status": 0
    },
    "create\_on": "2025-09-26T14:16:41.837-1100",
    "ret\_code": 0
}```
