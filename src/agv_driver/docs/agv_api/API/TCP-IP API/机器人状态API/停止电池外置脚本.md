**停止电池外置脚本**

**请求**

- 编号: 1902 (0x076E)
- 名称: robot\_status\_stopbatteryscript\_req
- 描述: 停止外置脚本
- JSON 数据区: 无

**请求示例**


```JSON
5A 01 00 00 00 00 00 00 07 6E 07 6E 00 00 00 00```

**响应**

- 编号: 11902 (0x2E7E)
- 名称: robot\_status\_stopbatteryscript\_res
- 描述: 停止外置脚本的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "create\_on": "2022-05-16T17:02:58.472Z",
  "ret\_code": 0
}```
