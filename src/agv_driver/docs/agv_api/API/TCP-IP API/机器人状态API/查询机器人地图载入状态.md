**查询机器人地图载入状态**

**请求**

- 编号: 1022 (0x03FE)
- 名称: robot\_status\_loadmap\_req
- 描述: 查询机器人当前地图载入状态
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 03 FE 00 00 00 00 00 00```

**响应**

- 编号: 11022 (0x2B0E)
- 名称: robot\_status\_loadmap\_res
- 描述: 查询机器人当前地图载入状态的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|loadmap\_status|number|0 = FAILED(载入地图失败), 1 = SUCCESS(载入地图成功), 2 = LOADING(正在载入地图)|是|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|


```<p>注意:</p><p>地图载入状态 (1022) 用于指示当前地图是否载入成功，当机器人刚启动或者发生切换地图操作时，地图载入状态会指示为 LOADING，此时无法进行重定位操作，必须等地图载入状态变为 SUCCESS 后才能进行重定位</p>```

**响应示例**


```JSON
{
  "loadmap\_status": 1
}```
