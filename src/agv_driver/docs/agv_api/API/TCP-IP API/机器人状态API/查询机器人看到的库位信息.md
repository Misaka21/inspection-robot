**查询机器人看到的库位信息**

版本：3.4.6.2+

**请求**

- 编号: 1803 (0x070B)
- 名称: robot\_status\_bins\_req
- 描述: 查询机器人能看到的库位状态信息
- JSON 数据区: 无

**响应**

- 编号: 11803 (0x71683)
- 名称: robot\_status\_bins\_res
- 描述：查询机器人看到的库位状态信息的响应
- JSON数据区：参见下面示例：

```JSON
{
    "header": {
        "pub\_nsec":1234,
        "data\_nsec":1234,
        "seq":1234,
        "frame\_id":"XXX"
    },
    "bins":[{
        "binId": "BIN-1",
        "filled": false,
        "status": 1  // 1表示断连，0表示连接
    }]
}```
