**查询跟随状态**

**请求**

- 编号: 1024(0x0400)
- 名称: robot\_status\_followw\_req
- 描述: 查询跟随状态
- JSON 数据区:无

**请求示例**

```JSON
5A 01 00 00 00 00 00 00 04 00 04 00 00 00 00 00```

**响应**

- 编号: 11024(0x2B10)
- 名称: robot\_status\_follow\_res
- 描述:查询标定状态的响应
- JSON 数据区: 见下表

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|target\_x|number|目标的 x 坐标|否|
|target\_y|number|目标的 y 坐标|否|
|tracking\_status|number|跟随状态|否|
**响应示例**

```JSON
{
    "create\_on": "2025-09-26T14:27:34.713+0800",
    "ret\_code": 0,
    "target\_x": 0,
    "target\_y": 0,
    "tracking\_status": 0
}```
