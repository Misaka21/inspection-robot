**查询建图时的3D二维码**

` `开发中 >3.4.4

**请求**

- 编号: 1665 (0x0681)
- 名称: robot\_status\_3dtag\_req
- 描述: 查询建图时的3D二维码
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|index|number|3D二维码数组中，起始的索引|否|
|num|number|查询的个数|否|
**请求示例**

暂停机械臂

```JSON
{"index":0, "num":4}```
**响应**

- 编号: 1 1665 (0x2D91)
- 名称: robot\_status\_3dtag\_res
- 描述: 查询建图时的3D二维码的响应
- JSON 数据区: 见下

```返回数据为 Message\_Map的json string，格式可能会有变化。```
