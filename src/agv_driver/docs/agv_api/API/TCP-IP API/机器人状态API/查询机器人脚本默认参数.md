**查询机器人脚本默认参数**

**请求**

- 编号: 1508 (0x05E4)
- 名称: robot\_status\_script\_args\_req
- 描述: 查询机器人脚本默认参数
- JSON 数据区: 见下表

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|type|string|机器人脚本的类型，“module”=机构脚本，“ genetic ”=通用脚本|否|
|relative\_path|string|脚本相对目录，与 [脚本查询 ](查询机器人脚本列表.md)中的相同|否|

**请求示例**

如下表示查询机器人机构脚本 syspy/addMoveTaskList.py的默认参数

```JSON
{
        "type": "module",
        "relative\_path": "syspy/addMoveTaskList.py"
}```

**响应**

- 编号: 11508 (0x2CF4)
- 名称: robot\_status\_script\_args\_res
- 描述: 查询机器人脚本默认参数 的响应
- JSON 数据区: 见下

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|default\_args|object|脚本默认参数|是|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|


```需注意，若参数说明和脚本说明中包含中文，则响应中会将中文转换成unicode编码字符串。需要用户手动转换成原文。```

**响应示例**


```JSON
{
        "create\_on": "2022-11-07T13:54:08.368Z",
        "default\_args": {
                "move\_task\_list": {
                        "tips": "\u4efb\u52a1",                                                                // unicode：任务
                        "type": "json",
                        "unit": "",
                        "value": ""
                }
        },
        "ret\_code": 0
}```
