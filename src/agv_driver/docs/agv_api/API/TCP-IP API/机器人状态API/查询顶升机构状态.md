**查询顶升机构状态**


```仅对选配顶升机构的机器人有意义```

**请求**

- 编号: 1027 (0x0403)
- 名称: robot\_status\_jack\_req
- 描述: 查询顶升机构状态
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 04 03 00 00 00 00 00 00```

**响应**

- 编号: 11027 (0x2B13)
- 名称: robot\_status\_jack\_res
- 描述: 查询顶升机构状态的响应
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|jack\_mode|boolean|运行模式, false = 手动, true = 自动|是|
|jack\_state|number|运行状态,  0x00 = 上升中，0x01 = 上升到位，0x02 = 下降中，0x03 = 下降到位，0x04 = 停止，0xFF = 执行失败|是|
|jack\_enable|boolean|顶升是否启用, false = 未启用, true = 启用|是|
|jack\_error\_code|number|顶升错误码|是|
|jack\_isFull|boolean|顶升机构上是否有料, false = 无料, true = 有料|是|
|jack\_speed|number|顶升机构电机速率, 单位: mm/s|是|
|jack\_emc|boolean|顶升机构是否急停, false = 未急停,  true = 急停|是|
|jack\_height|number|顶升当前的高度，单位: m|是|
|peripheral\_data|array[number]|外设中的用户自定义数据|是|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "jack\_emc": false,
  "jack\_enable": false,
  "jack\_error\_code": 0,
  "jack\_height": 0,
  "jack\_isFull": false,
  "jack\_mode": false,
  "jack\_speed": 0,
  "jack\_state": 0,
  "peripheral\_data": [],
  "ret\_code": 0
}```
