**查询机器人支持标定列表**

**请求**

- 编号: 1509
- 名称: ROBOT\_STATUS\_CALIB\_SUPPORT\_LIST\_REQ
- 描述: 查询机器人支持标定列表

**响应**

- 编号: 11509
- 名称: ROBOT\_STATUS\_CALIB\_SUPPORT\_LIST\_RES
- 描述: 查询机器人支持标定列表的响应
- JSON 数据区: 见下

```JSON
{
    "calib\_list":[
        {
            "calib\_type":1, //标定类型
            "device\_name":"laser1", //设备名
            "is\_auto":true,  //false:需要人工干预 true:不需要人工干预
            "text":"laser1", //提示字段
            "desc":"calib laser1", //提示字段
            "is\_calibrated":false,   //false:未标定 true:已标定
            "available":true, //是否可标定，true:是 false:否
            "priority\_type":0, //标定优先级，值越小优先级越高
            "layout":"SinSteer\_1", //标定项后面显示的提示消息
            "hasPlot":true //是否有标定图，true:有 false:无
        }
    ]
}```
