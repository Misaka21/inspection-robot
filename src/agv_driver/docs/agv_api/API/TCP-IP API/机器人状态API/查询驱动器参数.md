**查询驱动器参数**

**请求**

- 编号： 1750 （0x06D6）
- 名称： robot\_status\_canframe\_req
- 描述： 查询驱动器参数
- JSON 数据区： 无


```查询频率必须低于 20Hz！```

**请求示例**


```Plaintext
5A 01 00 00 00 00 00 00 06 D6 00 00 00 00 00 00 ```

**响应**

- 编号： 11750 （0x2DE6）
- 名称： robot\_status\_canframe\_res
- 描述： 查询驱动器参数的响应
- JSON 数据区： 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ID|uint32|发送报文的 canid，如 0x601|是|
|Extended|bool|报文是否为扩展型，一般为 false|是|
|DLC|uint32|发送报文的数据长度，一般为 8|是|
|Data|string|报文数据区|是|
|Channel|uint32|使用的端口，有 1 和 2|是|

**响应示例**


```JSON
{
  "ID": 601,
  "Extended": false,
  "Remote": false,
  "DLC": 8,
  "Data": "40 40 60 00 00 00 00 00",
  "Channel": 1
}```
