**查询货叉(叉车)状态**


```仅对有货叉的机器人有意义```

**请求**

- 编号: 1028 (0x0404)
- 名称: robot\_status\_fork\_req
- 描述: 查询货叉(叉车)状态
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 04 04 00 00 00 00 00 00 ```

**响应**

- 编号: 11028 (0x2B14)
- 名称: robot\_status\_fork\_res
- 描述: 查询货叉(叉车)状态的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|fork\_height|number|货叉高度, 单位 m|是|
|<p>fork\_height\_in\_place</p><p></p>|<p>boolean</p><p></p>|货叉高度是否到位, true = 到位, false = 未到位（ 弃用，目前该位随机器人任务状态而变化；当机器人的任务状态为完成时为true，当机器人运动时为false ）|是|
|<p>fork\_auto\_flag</p><p></p>|<p>boolean</p><p></p>|叉车的控制模式(通过叉车上的物理按钮切换), ture = 自动控制(控制器控制), false = 手动控制(方向盘驾驶)|是|
|forward\_val|number|货叉前移后退距离, 单位: m|是|
|forward\_in\_place|boolean|货叉前移后退是否到位, true = 到位, false = 未到位|是|
|fork\_pressure\_actual|number|货叉称重数据，单位：kg|是|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "fork\_auto\_flag": true,
  "fork\_height": 0,
  "fork\_height\_in\_place": false,
  "fork\_pressure\_actual": 0.0,
  "forward\_in\_place": false,
  "forward\_val": 0,
  "ret\_code": 0
}```
