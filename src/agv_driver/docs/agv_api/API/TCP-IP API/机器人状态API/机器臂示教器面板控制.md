**机器臂示教器面板控制**

**请求**

- 编号: 1674 (0x068A)
- 名称: robot\_status\_armoperation\_req
- 描述: 机器臂示教器面板控制
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|type|string|<p>机械臂示教器面板控制类型：</p><p>power\_on =上电</p><p>power\_off =下电</p><p>break\_release =松抱闸</p><p>shutdown=关机</p><p>unlock\_protective\_stop=解除保护性停止</p><p>gripper\_operation=控制夹爪开合</p><p></p>|否|
|<p>value</p><p></p>|<p>int</p><p></p>|夹爪的开合程度，[0, 100]，int类型。只有type为gripper\_operation才生效。|<p>否</p><p></p>|
**请求示例**

示教器上电

```JSON
{
    "type": "power\_on"
}```
**响应**

- 编号: 11674 (0x2D9A)
- 名称: robot\_status\_armoperation\_res
- 描述: 机器臂示教器面板控制的响应
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|
