**查询历史控制权信息**

**请求**

- 编号: 1061(0x0425)
- 名称: robot\_status\_lock\_history\_req
- 描述:查询历史控制权信息
- JSON 数据区:无

**请求示例**

```JSON
5A 01 00 00 00 00 00 00 04 25 04 25 00 00 00 00```

**响应**

- 编号: 11061(0x2B35)
- 名称: robot\_status\_lock\_history\_res
- 描述:查询历史控制权信息的响应
- JSON 数据区: 见下表

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|lock\_history|json array|历史控制权列表|否|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|
lock\_history 字段如下：

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ip|string|控制权所属ip|否|
|port|number|下发获取控制权请求的端口|否|
|type|number|控制权类型|否|
|nick\_name|string|控制权名称|否|
|time\_t|number|获取控制权的时间|否|
|desc|string|控制权描述|否|
|time\_t\_unlock|number|释放控制权的时间|否|
**响应示例**

```JSON
{
    "create\_on": "2025-09-28T18:05:30.890+0800",
    "lock\_history": [
        {
            "desc": "",
            "ip": "192.168.12.173",
            "nick\_name": "DESKTOP-80MB6O0",
            "port": 57426,
            "time\_t": 1759053921,
            "time\_t\_unlock": 1759053928,
            "type": 2
        }
    ],
    "ret\_code": 0
}```
