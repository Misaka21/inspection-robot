**查询机器人任务列表**

**请求**

- 编号: 1026(0x0402)
- 名称: robot\_status\_tasklist\_req
- 描述:查询机器人任务列表
- JSON 数据区:无

**请求示例**

```JSON
5A 01 00 00 00 00 00 00 04 02 04 02 00 00 00 00```

**响应**

- 编号: 11026(0x2B12)
- 名称: robot\_status\_tasklist\_res
- 描述:查询机器人任务列表的响应
- JSON 数据区: 见下表

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|tasklist\_status|json|任务列表状态|否|
**响应示例**

```JSON
{
    "create\_on": "2025-09-26T14:34:23.191+0800",
    "ret\_code": 0,
    "tasklist\_status": {
        "actionGroupId": 0,
        "actionIds": [
        ],
        "loop": false,
        "taskId": 0,
        "taskListName": "",
        "taskListStatus": 0
    }
}```
