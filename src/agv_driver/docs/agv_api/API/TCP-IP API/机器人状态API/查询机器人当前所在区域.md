**查询机器人当前所在区域**

**请求**

- 编号: 1011 (0x03F3)
- 名称: robot\_status\_area\_req
- 描述: 查询机器人当前所在的区域信息
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 03 F3 00 00 00 00 00 00```

**响应**

- 编号: 11011 (0x2B03)
- 名称: robot\_status\_area\_res
- 描述: 查询机器人当前所在的区域信息的响应
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|area\_ids|array[string]|机器人所在区域 id 的数组(由于地图上的区域是可以重叠的, 所以机器人可能同时在多个区域)，数组可能为空|是|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**

如上说明机器人当前处于 id 号为 "1" 和 "2" 的区域中

```JSON
{
  "area\_ids": [
    "1",
    "2"
  ]
}```
