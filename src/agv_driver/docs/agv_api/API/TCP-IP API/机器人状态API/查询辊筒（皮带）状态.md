**查询辊筒（皮带）状态**


```仅对选配有辊筒/皮带的机器人有意义```

**请求**

- 编号: 1029 (0x0405)
- 名称: robot\_status\_roller\_req
- 描述: 查询辊筒(皮带)状态
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 04 05 00 00 00 00 00 00```

**响应**

- 编号: 11029 (0x2B15)
- 名称: robot\_status\_roller\_res
- 描述: 查询辊筒(皮带)状态的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|roller\_mode|boolean|运行模式, false = 手动, true = 自动|是|
|roller\_state|number|运行状态, 0x01 = 待机, 0x02 = 执行中, 0x03 = 执行完成, 0x04 = 停止, 0xFF = 执行失败|是|
|roller\_enable|boolean|辊筒是否启用, false = 未启用, true = 启用|是|
|roller\_error\_code|number|辊筒错误码|是|
|roller\_isFull|boolean|辊筒上是否有料, false = 无料, true = 有料|是|
|roller\_speed|number|辊筒电机速率|是|
|roller\_emc|boolean|辊筒是否急停, false = 未急停, true = 急停|是|
|peripheral\_data|array[number]|外设中的用户自定义数据|是|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**roller\_error\_code 定义**


```JSON
0x00（00）,       // 无错误
0x11（17）,       // 电机错误
0x01（01）,       // 辊筒已有物料（上料）
0x02（02）,       // 辊筒无物料（下料）
0x03（03）,       // 上料超时
0x04（04）,       // 下料超时
0x05（05）,       // 物料超长```

**响应示例**


```JSON
{
  "peripheral\_data": [],
  "ret\_code": 0,
  "roller\_emc": false,
  "roller\_enable": false,
  "roller\_error\_code": 0,
  "roller\_isFull": false,
  "roller\_mode": false,
  "roller\_speed": 0,
  "roller\_state": 0
}```
