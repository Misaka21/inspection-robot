**查询障碍物信息**

**请求**

- 编号: 1019(0x03FB)
- 名称: robot\_status\_obstacle\_req
- 描述: 查询障碍物信息
- JSON 数据区:无

**请求示例**

```JSON
5A 01 00 00 00 00 00 00 03 FB 03 FB 00 00 00 00```

**响应**

- 编号: 11019(2B0B)
- 名称: robot\_status\_obstacle\_res
- 描述: 查询障碍物信息的响应
- JSON 数据区: 见下表

```字段名|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|user\_objects|Json array|障碍物信息|否|
