**圆弧运动**

**请求**

- 编号: 3058 (0x0BF2)
- 名称: robot\_task\_circular\_req
- 描述: 圆弧运动
- JSON 数据区: 见下表

```<p>注意:</p><p>3055(平动), 3056(转动), 3057, 3058 不能同时进行</p>```


```字段名|类型|描述|可缺省``` :- | :- | :- |
|rot\_radius|number|旋转半径, 单位m; 圆心在机器人坐标系左边为正, 右边为负|是|
|rot\_degree|number|旋转角度, 单位度|是|
|rot\_speed|number|旋转速度, 单位为 rad/s, 如果是负数, 则说明是顺时针走|是|
|mode|number|0 = 里程模式(根据里程进行运动), 1 = 定位模式, 若缺省则默认为里程模式|是|

**请求示例**


```JSON
{"rot\_radius":1,"rot\_degree":360,"rot\_speed":0.5, "mode":0}
// 以旋转半径1m, 0.5 rad/s 旋转 360 度```

**响应**

- 编号: 13058 (0x3302)
- 名称: robot\_task\_circular\_res
- 描述: 圆弧运动
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```
