**查询机器人所有任务链**

**请求**

- 编号: 3115 (0x0C2B)
- 名称: robot\_tasklist\_list\_req
- 描述: 查询机器人所有任务链
- JSON 数据区: 无

**请求示例**

无

**响应**

- 编号: 13115 (0x333B)
- 名称: robot\_tasklist\_list\_res
- 描述: 查询机器人所有任务链的响应
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|tasklists|string array|任务链列表|是|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "create\_on": "2022-11-24T17:57:37.153Z",
  "ret\_code": 0,
  "tasklists": ["21-01-04-B-08-01-01", "task\_20210720020304790"]
}```
