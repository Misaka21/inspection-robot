**执行预存任务链**

**请求**

- 编号: 3106 (0x0C22)
- 名称: robot\_tasklist\_name\_req
- 描述: 执行预存任务链
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|name|string|预存的任务链的名称(使用 Roboshop 上传)|否|

**请求示例**


```JSON
{"name":"task\_666"}
// 执行预存在机器人内名为task\_666的任务链
5A 01 00 01 00 00 00 13 0C 22 00 00 00 00 00 00 
7B 22 6E 61 6D 65 22 3A 22 74 61 73 6B 5F 36 36 
36 22 7D```

**响应**

- 编号: 13106 (0x3332)
- 名称: robot\_tasklist\_name\_res
- 描述: 执行预存任务链的响应
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 33 32 0C 22 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
