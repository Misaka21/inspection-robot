**库位检测**

**请求**

- 编号: 6901
- 名称: robot\_other\_bin\_detect\_req
- 描述: 触发一次库位检测
- JSON 数据区: 如下

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|seq|number|时间戳/uuid|否|
**请求示例**


```JSON
{
  "seq": 26446717
}```
注：该功能实际上将存储库位监测结果并发布Message\_Bin

**响应**

- 编号: 16901
- 名称: robot\_other\_bin\_detect\_res
- 描述: 库位检测是否正确调用服务
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```
