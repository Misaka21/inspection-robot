**辊筒(皮带)前后反向**


```<p>注意：</p><p>使用该指令设置反向后，辊筒的前后逻辑会相反，即前上料变为后上料，后上料变为前上料。</p>```

**请求**

- 编号: 6069 (0x17B5)
- 名称: robot\_other\_roller\_front\_back\_inverse\_req
- 描述: 辊筒(皮带)前后反向
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|inverse|boolean|是否反向，true = 反向, false = 不反向(即恢复正常)|否|

**请求示例**


```JSON
5A 01 00 01 00 00 00 10 17 B5 00 00 00 00 00 00
{"inverse":true}
// 辊筒(皮带)前后反向
7B 22 69 6E 76 65 72 73 65 22 3A 74 72 75 65 7D```

**响应**

- 编号: 16069 (0x3EC5)
- 名称: robot\_other\_roller\_front\_back\_inverse\_res
- 描述: 辊筒(皮带)前后反向的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 3E C5 17 B5 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
