**辊筒(皮带)左右反向**

```<p>注意：</p><p>使用该指令设置反向后，辊筒的左右逻辑会相反，即左上料变为右上料，右上料变为左上料。</p>```

**请求**

- 编号: 6068 (0x17B4)
- 名称: robot\_other\_roller\_left\_right\_inverse\_req
- 描述: 辊筒(皮带)左右反向
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|inverse|boolean|是否反向，true = 反向, false = 不反向(即恢复正常)|否|

**请求示例**


```JSON
5A 01 00 01 00 00 00 10 17 B4 00 00 00 00 00 00
{"inverse":true}
// 辊筒(皮带)左右反向
7B 22 69 6E 76 65 72 73 65 22 3A 74 72 75 65 7D```

**响应**

- 编号: 16068 (0x3EC4)
- 名称: robot\_other\_roller\_left\_right\_inverse\_res
- 描述: 辊筒(皮带)左右反向的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 3E C4 17 B4 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
