**电机使能与去使能**

**请求**

- 编号: 6201 (0x1839)
- 名称: robot\_other\_set\_motor\_enable\_req
- 描述: 电机使能与去使能
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|motor\_name|string|电机名|否|
|enable|boolean|true=使能; false=去使能|否|

**请求示例**


```JSON
5A 01 00 01 00 00 00 26 18 39 00 00 00 00 00 00
// 使名为 motor1 的电机去使能：
{"motor\_name":"motor1","enable":false}
// 序列化后十六进制为：
7B 22 6D 6F 74 6F 72 5F 6E 61 6D 65 22 3A 22 6D 
6F 74 6F 72 31 22 2C 22 65 6E 61 62 6C 65 22 3A 
66 61 6C 73 65 7D```

**响应**

- 编号: 16201 (0x3F49)
- 名称: robot\_other\_set\_motor\_enable\_res
- 描述: 电机使能与去使能的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 3F 49 18 39 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
