**更新透传数据**

**请求**

- 编号: 6900
- 名称: robot\_other\_update\_transparent\_data\_req
- 描述: 更新透传数据
- JSON 数据区: 如下

**请求示例**


```JSON
{
  "key": {...} // key 为用户自定义的字段, value 为用户自定义的数据内容
}```

- 编号: 16900
- 名称: robot\_other\_update\_transparent\_data\_res
- 描述: 更新透传数据的响应

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```
