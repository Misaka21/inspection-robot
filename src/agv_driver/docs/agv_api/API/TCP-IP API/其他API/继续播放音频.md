**继续播放音频**


```<p>注意：</p><p>将从暂停处继续播放。</p>```

**请求**

- 编号: 6011 (0x177B)
- 名称: robot\_other\_resume\_audio\_req
- 描述: 继续播放音频
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 17 7B 00 00 00 00 00 00```

**响应**

- 编号: 16011 (0x3E8B)
- 名称: robot\_other\_resume\_audio\_res
- 描述: 继续播放音频的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 3E 8B 17 7B 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
