**辊筒(皮带)向右滚动**


```<p>注意：</p><p>滚动时两侧挡板均放下。</p>```


```<p></p><p>注意：</p><p>该指令会立即返回, 需通过 1029 或 1100 指令中的 roller\_state 来查询滚动是否完成。滚动不会因超时而失败，辊筒会一直转动，直到接收新的指令。</p>```


```<p>警告：</p><p>机器人运动过程中不接受辊筒运动指令。</p>```

**请求**

- 编号: 6054 (0x17A6)
- 名称: robot\_other\_roller\_right\_roll\_req
- 描述: 辊筒(皮带)向右滚动
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 17 A6 00 00 00 00 00 00```

**响应**

- 编号: 16054 (0x3EB6)
- 名称: robot\_other\_roller\_right\_roll\_res
- 描述: 辊筒(皮带)向右滚动的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 3E B6 17 A6 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
