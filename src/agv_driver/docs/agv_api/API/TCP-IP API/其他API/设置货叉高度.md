**设置货叉高度**


```<p>注意：</p><p>该 API 用于设置叉车的货叉高度, 该指令会立即返回, 需通过 1028 或 1100 指令中的 fork\_height\_in\_place 来查询货叉是否到位。</p>```

**请求**

- 编号: 6040 (0x1798)
- 名称: robot\_other\_set\_fork\_height\_req
- 描述: 设置货叉高度
- JSON 数据区: 见下表


```<p>警告：</p><p>机器人运动过程中不接受辊筒运动指令。</p>```


```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|height|number|货叉高度, 单位 m<br>针对地牛：1=上升；0=下降|否|

**请求示例**


```JSON
5A 01 00 01 00 00 00 0E 17 98 00 00 00 00 00 00
{"height":1.2}
// 序列化后十六进制为：
7B 22 68 65 69 67 68 74 22 3A 31 2E 32 7D```

**响应**

- 编号: 16040 (0x3EA8)
- 名称: robot\_other\_set\_fork\_height\_res
- 描述: 设置货叉高度的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 3E A8 17 98 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
