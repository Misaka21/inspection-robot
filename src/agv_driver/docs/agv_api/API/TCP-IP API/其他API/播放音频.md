**播放音频**

**请求**

- 编号: 6000 (0x1770)
- 名称: robot\_other\_play\_audio\_req
- 描述: 播放音频, 播放指定的音频文件
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|name|string|需要播放的音频文件名(不含文件扩展名)|否|
|loop|boolean|是否循环播放(true = 循环, false = 不循环,如果缺省则默认不循环)|是|

**请求示例**


```JSON
5A 01 00 01 00 00 00 20 17 70 00 00 00 00 00 00
{"name":"collision","loop":true}
// 循环播放collision
7B 22 6E 61 6D 65 22 3A 22 63 6F 6C 6C 69 73 69 
6F 6E 22 2C 22 6C 6F 6F 70 22 3A 74 72 75 65 7D```

**响应**

- 编号: 16000 (0x3E80)
- 名称: robot\_other\_play\_audio\_res
- 描述: 播放音频的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 3E 80 17 70 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
