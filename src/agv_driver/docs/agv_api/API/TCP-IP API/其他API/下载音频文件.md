**下载音频文件**

**请求**

- 编号: 6031 (0x178F)
- 名称: robot\_other\_download\_audio\_req
- 描述: 下载音频文件
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|name|string|需要下载的音频文件名(不含文件扩展名)|否|

**请求示例**


```JSON
5A 01 00 01 00 00 00 0E 17 8F 00 00 00 00 00 00
{"name":"collision"}
// 下载collision
// 序列化后的十六进制数据：
7B 22 6E 61 6D 65 22 3A 22 63 6F 6C 6C 69 73 69 6F 6E 22 7D```

**响应**

- 编号: 16031 (0x3E9F)
- 名称: robot\_other\_download\_audio\_res
- 描述: 下载音频文件的响应
- JSON 数据区: 见下表

如果不发生错误，数据区中的内容为文件内容

若发生错误，则响应示例如下：

**响应示例**


```JSON
{
    "create\_on": "2021-11-10T15:45:30.077Z",
    "err\_msg": "audio file collision not exists",
    "ret\_code": 40060
}```
