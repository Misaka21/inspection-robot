**停止扫描地图**

**请求**

- 编号: 6101 (0x17D5)
- 名称: robot\_other\_endslam\_req
- 描述: 停止扫描地图
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 17 D5 00 00 00 00 00 00```

**响应**

- 编号: 16101 (0x3EE5)
- 名称: robot\_other\_endslam\_res
- 描述: 停止扫描地图的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|real\_time|bool|是否开启实时扫图数据传输|是|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "real\_time": true,
  "ret\_code": 0
}```
