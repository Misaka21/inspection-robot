**暂停播放音频**

**请求**

- 编号: 6010 (0x177A)
- 名称: robot\_other\_pause\_audio\_req
- 描述: 暂停播放音频
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 17 7A 00 00 00 00 00 00```

**响应**

- 编号: 16010 (0x3E8A)
- 名称: robot\_other\_pause\_audio\_res
- 描述: 暂停播放音频的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 3E 8A 17 7A 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
