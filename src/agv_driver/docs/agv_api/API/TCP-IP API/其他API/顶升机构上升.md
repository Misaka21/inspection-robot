**顶升机构上升**


```<p>注意：</p><p>该指令会立即返回, 需通过 1027 或 1100 指令中的 jack\_state 来查询顶升状态。</p>```

**请求**

- 编号: 6070 (0x17B6)
- 名称: robot\_other\_jack\_load\_req
- 描述: 顶升机构上升
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 17 B6 00 00 00 00 00 00```

**响应**

- 编号: 16070 (0x3EC6)
- 名称: robot\_other\_jack\_load\_res
- 描述: 顶升机构上升的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 3E C6 17 B6 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
