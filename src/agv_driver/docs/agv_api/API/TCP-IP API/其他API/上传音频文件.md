**上传音频文件**

**请求**

- 编号: 6030 (0x178E)
- 名称: robot\_other\_upload\_audio\_req
- 描述: 上传音频文件
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|文件二进制数据|文件二进制数据|文件二进制数据|否|


```<p>文件二进制数据最后二十位为文件名（不含后缀），要求文件名只含有以下字符：</p><p>- 小写字母（a-z）</p><p>- 大写字母（A-Z）</p><p>- 数字（0-9）</p><p>- 下划线（\_）</p><p>- 破折号（-）</p><p>- 圆括号（()）</p><p>- 方括号（[]）</p><p>- 小于号（<）</p><p>- 大于号（>）</p><p>- 空格（\s）</p><p>文件二进制数据其余部分为音频文件二进制数据。</p>```


```示例程序可参考 [上传机器人文件 ](..\机器人状态API\上传机器人文件.md)。```

**响应**

- 编号: 16030 (0x3E9E)
- 名称: robot\_other\_upload\_audio\_res
- 描述: 上传音频文件的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 3E 9E 17 70 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
