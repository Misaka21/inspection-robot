**写入外设用户自定义数据**

**请求**

- 编号: 6049 (0x17A1)
- 名称: robot\_other\_write\_peripheral\_data\_req
- 描述: 写入外设用户自定义数据（对于hook、jack、roller）三类外设
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|id|uint32\_t|id 号,  0~15|否|
|data|uint16\_t|modbus寄存器数据|否|

**请求示例**

单个写入


```JSON
5A 01 00 01 00 00 00 13 17 A1 00 00 00 00 00 00
{"id":0,"data":233}
// 序列化后十六进制为：
7B 22 69 64 22 3A 30 2C 22 64 61 74 61 22 3A 32 
33 33 7D```

多个写入


```JSON
{
  "peripheral\_data": [{
    "id": 1,
    "data": 1
  },
  {
    "id": 2,
    "data": 2
  }]
}```

**响应**

- 编号: 16049 (0x3EB1)
- 名称: robot\_other\_write\_peripheral\_data\_res
- 描述: 写入外设用户自定义数据的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 3E B1 17 A1 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
