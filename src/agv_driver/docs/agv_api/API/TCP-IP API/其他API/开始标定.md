**开始标定**

**请求**

- 编号: 6110
- 名称: ROBOT\_OTHER\_CALIBRATE\_REQ
- 描述: 开始标定
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|calib\_list|object array|需要标定的设备列表|否|

**请求示例**


```JSON
{
    "calib\_list":[
        {
            "calib\_type":1, //标定类型
            "device\_name":"laser1" //设备名
        }
    ]
}```

注1：标定类型需要查看标定类型列表，见：[查询机器人支持标定列表](..\机器人状态API\查询机器人支持标定列表.md)

注2：设备名需要查看模型文件中的设备名称

**响应**

- 编号: 16110
- 名称: ROBOT\_OTHER\_CALIBRATE\_RES
- 描述: 开始标定的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|
