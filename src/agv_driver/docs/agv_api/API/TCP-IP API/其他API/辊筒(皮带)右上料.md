**辊筒(皮带)右上料**


```<p>注意：</p><p>该指令会立即返回, 需通过 1029 或 1100 指令中的 roller\_state 来查询上料是否完成。右上料可能会因超时而失败。</p>```


```<p>警告：</p><p>机器人运动过程中不接受辊筒运动指令。</p>```

- 编号: 6063 (0x17AF)
- 名称: robot\_other\_roller\_right\_load\_req
- 描述: 辊筒(皮带)右上料
- JSON 数据区: 无

```Plaintext
