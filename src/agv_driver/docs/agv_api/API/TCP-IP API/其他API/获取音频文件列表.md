**获取音频文件列表**

**请求**

- 编号: 6033 (0x1791)
- 名称: robot\_other\_audio\_list\_req
- 描述: 获取音频文件列表
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 17 91 00 00 00 00 00 00```

**响应**

- 编号: 16033 (0x3EA1)
- 名称: robot\_other\_audio\_list\_res
- 描述: 获取音频文件列表的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|audios|array[string]|所有储存在机器人上的音频文件名字组成的数组|否|
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|
