**设置继电器**


```<p>注意：</p><p>该 API 用于设置机器人控制器上的继电器状态。（目前只有SRC2000控制器有该接口）</p>```

**请求**

- 编号: 6003 (0x1773)
- 名称: robot\_other\_setrelay\_req
- 描述: 设置继电器
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|status|boolean|true 为继电器吸合, false 为继电器断开|否|

**请求示例**


```JSON
5A 01 00 01 00 00 00 0F 17 73 00 00 00 00 00 00
{"status":true}
// 设置继电器为吸合
// 序列化后十六进制为：
7B 22 73 74 61 74 75 73 22 3A 74 72 75 65 7D```

**响应**

- 编号: 16003 (0x3E83)
- 名称: robot\_other\_setrelay\_res
- 描述: 设置继电器的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 3E 83 17 73 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
