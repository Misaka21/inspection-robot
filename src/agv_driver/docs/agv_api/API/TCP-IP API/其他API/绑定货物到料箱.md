**绑定货物到料箱**


```<p>注意：</p><p>此 API 仅对料箱车有效，PS：料箱车指在机器人模型文件中配置了 container 的车。</p>```


```<p>注意：</p><p>有关料箱车 container 详细信息查询可以参考， [机器人导航状态 ](..\机器人状态API\查询机器人导航状态.md)或者 [批量数据1 ](..\机器人状态API\查询批量数据1.md)。</p>```

**请求**

- 编号: 6804 (0x1A94)
- 名称: robot\_other\_set\_container\_goods\_req
- 描述: 绑定货物到料箱
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|goods\_id|string|货物id|否|
|container\_name|string|料箱名（在模型文件中配置container）|否|
|desc|string|货物描述|否|

**请求示例**


```JSON
5A 01 00 00 00 00 00 3D 1A 94 1A 94 00 00 00 00
// 绑定货物 good1 到名为 1 的料箱：
{"goods\_id":"good1","container\_name":"1","desc":"first good"}
// 序列化后十六进制为：
7B 22 67 6F 6F 64 73 5F 69 64 22 3A 22 67 6F 6F
64 31 22 2C 22 63 6F 6E 74 61 69 6E 65 72 5F 6E
61 6D 65 22 3A 22 31 22 2C 22 64 65 73 63 22 3A
22 66 69 72 73 74 20 67 6F 6F 64 22 7D```

**响应**

- 编号: 16804 (0x41A4)
- 名称: robot\_other\_set\_container\_goods\_res
- 描述: 绑定货物到料箱的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "create\_on": "2018-01-29T04:09:34.009Z",
  "ret\_code": 0
}```
