**取消重定位**


```用于取消重定位过程, 如果当前正在重定位(即 reloc\_status = 2 时)将取消本次重定位, 机器人的定位位置将回到发送重定位指令前的位置,并且 reloc\_status 将变为 3 (重定位完成)。 如果当前没有正在重定位, 则本条指令没有任何作用。```

**请求**

- 编号: 2004 (0x07D4)
- 名称: robot\_control\_cancelreloc\_req
- 描述: 取消重定位
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 07 D4 00 00 00 00 00 00```

**响应**

- 编号: 12004 (0x2EE4)
- 名称: robot\_control\_cancelreloc\_res
- 描述: 取消重定位的响应
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```
