**电机编码器标零**

**请求**

- 编号: 2024 (0x07E8)
- 名称: robot\_control\_clearmotorencoder\_req
- 描述: 对指定电机标零
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|name|string|要标零的电机名|否|

**请求示例**


```JSON
5A 01 00 00 00 00 00 0C 07 E8 00 00 00 00 00 00
{"name":"a"}
// 序列化后十六进制为:
7B 22 6E 61 6D 65 22 3A 22 61 22 7D```

**响应**

- 编号: 12024 (0x2EF8)
- 名称: robot\_control\_clearmotorencoder\_res
- 描述: 对指定电机标零的响应
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|ret\_code|number|API 错误码|否|
|create\_on|string|API 上传时间戳|否|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "create\_on": "2021-11-08T17:25:00.824Z",
  "ret\_code": 0
}```
