**重定位**

**请求**

- 编号: 2002 (0x07D2)
- 名称: robot\_control\_reloc\_req
- 描述: 重定位
- JSON 数据区: 见下表

**注：不可所有参数都缺省**

```字段名|类型|描述|可缺省``` :- | :- | :- |
|isAuto|bool|是否为自动重定位，当存在该字段且值为true，忽略以下所有字段|是|
|x|number(double)|世界坐标系中的 x 坐标, 单位 m|否|
|y|number(double)|世界坐标系中的 y 坐标, 单位 m|否|
|angle|number(double)|世界坐标系中的角度, 单位 rad|是|
|length|number(double)|重定位区域半径，单位 m|是|
|home|<p>bool</p><p></p>|在 RobotHome 重定位(若为 true, 前三个参数无效, 并从 Roboshop 参数配置中的 RobotHome1-5 重定位, 若 RobotHome1-5 未配置, 则不做任何操作。若缺省则认为是 false)|是|

**请求示例**


```JSON
5A 01 00 01 00 00 00 1C 07 D2 00 00 00 00 00 00 
{"x":10.0,"y":3.0,"angle":0}
//序列化后十六进制为：
7B 22 78 22 3A 31 30 2E 30 2C 22 79 22 3A 33 2E 30 2C 22 61 6E 67 6C 65 22 3A 30 7D```

**响应**

- 编号: 12002 (0x2EE2)
- 名称: robot\_control\_reloc\_res
- 描述: 重定位的响应
- JSON 数据区: 见下表

```字段名|类型|描述|可缺省``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```Plaintext
5A 01 00 01 00 00 00 00 2E E2 00 00 00 00 00 00```
