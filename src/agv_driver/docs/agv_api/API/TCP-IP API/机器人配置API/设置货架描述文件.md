**设置货架描述文件**

**请求**

- 编号: 4357 (0x1105)
- 名称: robot\_config\_set\_shelfshape\_req
- 描述: 设置当前使用的货架描述文件
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|object\_path|string|货架描述文件相对路径<br>(不能包含中文等非法字符, 只能使用 0-9, a-z, A-Z, -, \_)|否|

**请求示例**


```JSON
5A 01 00 00 00 00 00 23 11 05 00 00 00 00 00 00
{"object\_path":"shelf/s0002.shelf"}
//序列化后十六进制为：
7B 22 6F 62 6A 65 63 74 5F 70 61 74 68 22 3A 22 
73 68 65 6C 66 2F 73 30 30 30 32 2E 73 68 65 6C 
66 22 7D```

**响应**

- 编号: 14357 (0x3815)
- 名称: robot\_config\_set\_shelfshape\_res
- 描述: 设置货架描述文件的响应
- JSON 数据区: 见下

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 38 15 11 05 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
