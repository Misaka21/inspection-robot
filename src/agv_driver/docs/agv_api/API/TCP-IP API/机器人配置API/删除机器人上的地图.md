**删除机器人上的地图**

**请求**

- 编号: 4012 (0x0FAC)
- 名称: robot\_config\_removemap\_req
- 描述: 删除机器人上的地图
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|map\_name|string|地图名(不能包含中文等非法字符, 只能使用 0-9, a-z, A-Z, -, \_)|否|

**请求示例**


```JSON
{"map\_name":"20210126173805395"}
// 删除机器人上名为20210126173805395的地图
5A 01 00 01 00 00 00 20 0F AC 00 00 00 00 00 00
7B 22 6D 61 70 5F 6E 61 6D 65 22 3A 22 32 30 32 
31 30 31 32 36 31 37 33 38 30 35 33 39 35 22 7D```

**响应**

- 编号: 14012 (0x36BC)
- 名称: robot\_config\_removemap\_res
- 描述: 删除机器人上的地图的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 36 BC 0F AC 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
