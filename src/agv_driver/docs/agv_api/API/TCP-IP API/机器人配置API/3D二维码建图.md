**3D二维码建图**

开发中  >3.4.3

**请求**

- 编号: 4353 (0x1101)
- 名称: robot\_config\_3dtagmapping\_req
- 描述: 3D二维码建图
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|start|bool|true=开启建图，false=关闭建图|否|

**请求示例**


```JSON
{"start":true}```

**响应**

- 编号: 14353 (0x3811)
- 名称: robot\_config\_3dtagmapping\_res
- 描述: 3D二维码建图的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```
