**恢复机器人参数为默认值**


```<p>注意：</p><p>请参考 [查询机器人参数 ](..\机器人状态API\查询机器人参数.md)，来设置需要恢复默认值的参数。</p>```

**请求**

- 编号: 4102 (0x1006)
- 名称: robot\_config\_reloadparams\_req
- 描述: 恢复机器人参数为默认值
- JSON 数据区: 见下表


```<p>注意：</p><p>请求数据为 JSON object array，其中每一个 object 代表一个插件需要恢复为默认值的参数，若请求数据为空 JSON object array，则恢复所有插件所有参数为默认值。</p>```


```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|params|string array|需要恢复为默认值的参数列表，若为空，则恢复当前插件所有参数为默认值|是|
|plugin|string|插件名|是|

**恢复插件 NetProtocol 和插件 MoveFactory 中相关参数为默认值**

**请求示例**


```JSON
[{
  "params": ["RobotNote" // 参数名
  ],
  "plugin": "NetProtocol" // 插件名
},
{
  "params": ["3DCameraHole"],
  "plugin": "MoveFactory"
}]```

**响应**

- 编号: 14102 (0x3716)
- 名称: robot\_config\_reloadparams\_res
- 描述: 恢复机器人参数为默认值的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "create\_on":"2024-05-07T13:49:33.208+0800",
  "ret\_code":0
}```
