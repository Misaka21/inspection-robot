**设置标定过程数据**

**请求**

- 编号: 4201
- 名称: ROBOT\_CONFIG\_CALIB\_PUSH\_DATA\_REQ
- 描述: 设置标定过程数据
- JSON 数据区: 见下表

**请求示例**

```JSON
{
  "data":"Fitting error during the process is 0.022724m per meter movement (in wheel movement meaning)"
}```
注：输入的数据仅起提示作用

**响应**

- 编号: 14201
- 名称: ROBOT\_CONFIG\_CALIB\_PUSH\_DATA\_RES
- 描述: 设置标定过程数据的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|
