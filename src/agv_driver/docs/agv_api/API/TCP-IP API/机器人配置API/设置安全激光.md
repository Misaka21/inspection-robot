**设置安全激光**

**请求**

- 编号: 4451(0x1163)
- 名称: robot\_config\_set\_safe\_laser\_req
- 描述: 设置安全激光
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|key|string|激光的key|否|
|enable|bool|是否启用|否|

**请求示例**

```JSON
{
    "key":"Laser-000",
    "enable":true
}```

**响应**

- 编号: 14451(0x3873)
- 名称: robot\_config\_set\_safe\_laser\_res
- 描述: 设置安全激光的响应
- JSON 数据区: 见下

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**

```JSON
{
  "create\_on": "2024-05-07T13:54:38.796+0800",
  "ret\_code": 0
}```
