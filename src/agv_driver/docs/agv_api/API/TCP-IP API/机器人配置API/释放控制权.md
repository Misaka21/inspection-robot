**释放控制权**



```<p>注意:</p><p>只能释放自己抢占的控制权，不能释放别人的控制权，如果想取得控制权，须抢占</p>```

**请求**

- 编号: 4006 (0x0FA6)
- 名称: robot\_config\_unlock\_req
- 描述: 释放控制权
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 0F A6 00 00 00 00 00 00```

**响应**

- 编号: 14006 (0x36B6)
- 名称: robot\_config\_unlock\_res
- 描述: 释放控制权的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 36 B6 0F A6 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
