**插入动态障碍物(机器人坐标系)**

**请求**

- 编号: 4350 (0x10FE)
- 名称: robot\_config\_addobstacle\_req
- 描述: 插入动态障碍物(机器人坐标系)
- JSON 数据区: 见下表

```<p>注意：</p><p>插入机器人坐标系下的矩形动态障碍物。</p>```


```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|name|string|障碍物名称, 需要唯一|否|
|x1|double|矩形第一个顶点的 x 坐标(机器人坐标系), 单位 m|否|
|y1|double|矩形第一个顶点的 y 坐标(机器人坐标系), 单位 m|否|
|x2|double|矩形第二个顶点的 x 坐标(机器人坐标系), 单位 m|否|
|y2|double|矩形第二个顶点的 y 坐标(机器人坐标系), 单位 m|否|
|x3|double|矩形第三个顶点的 x 坐标(机器人坐标系), 单位 m|否|
|y3|double|矩形第三个顶点的 y 坐标(机器人坐标系), 单位 m|否|
|x4|double|矩形第四个顶点的 x 坐标(机器人坐标系), 单位 m|否|
|y4|double|矩形第四个顶点的 y 坐标(机器人坐标系), 单位 m|否|

- 名称: robot\_config\_addobstacle\_res
