**配置驱动器参数**



```<p>注意:</p><p>请参考 [查询驱动器参数 ](..\机器人状态API\查询驱动器参数.md)，来设置参数</p>```
**请求**

- 编号: 4400 (0x1130)
- 名称: robot\_config\_send\_canframe\_req
- 描述: 配置驱动器参数
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|channel|uint32|使用的端口，有1和2|是|
|id|uint32|发送报文的 canid，如0x601|是|
|dlc|uint32|发送报文的数据长度，一般为 8|是|
|extend|bool|报文是否为扩展型，一般为 false|是|
|data|string|报文数据区，如 "40 40 60 00 00 00 00 00"，十六进制，空格隔开|是|

**请求示例**

```JSON
{
  "channel": 1,
  "id": 601,
  "dlc": 8,
  "extend": false,
  "data":  "40 40 60 00 00 00 00 00"
}```

**响应**

- 编号: 14400 (0x3840)
- 名称: robot\_config\_send\_canframe\_res
- 描述: 配置驱动器参数的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**

```JSON
{
  "create\_on": "2024-05-07T13:22:53.975+0800",
  "ret\_code": 0
}```
