- 名称: robot\_config\_joystick\_bind\_keymap\_res
- 描述: 上传手柄自定义绑定事件的响应
- JSON 数据区: 见下

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |

|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|
