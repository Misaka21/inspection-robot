**上传模型文件到机器人**

**请求**

- 编号: 4200 (0x1068)
- 名称: robot\_config\_model\_req
- 描述: 上传模型文件到机器人
- JSON 数据区: 见下

```<p>注意：</p><p>数据区中的内容为 json 格式的模型文件, 模型文件格式详情请参见 [模型文件格式 ](..\附录\模型文件格式.md)。</p>```

**请求示例**

**[robot.model]**

**响应**

- 编号: 14200 (0x3778)
- 名称: robot\_config\_model\_res
- 描述: 上传模型文件到机器人的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "create\_on":"2024-05-07T13:38:57.881+0800",
  "ret\_code":0
} ```
