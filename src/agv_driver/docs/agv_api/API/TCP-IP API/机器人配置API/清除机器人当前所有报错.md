**清除机器人当前所有报错**

**请求**

- 编号: 4009 (0x0FA9)
- 名称: robot\_config\_clearallerrors\_req
- 描述: 清除机器人当前所有报错（仅清除error级别的错误）
- JSON 数据区: 无

**请求示例**


```Plaintext
5A 01 00 01 00 00 00 00 0F A9 00 00 00 00 00 00```

清除指定报错码


```JSON
{"error\_codes":[52000]}```

**响应**

- 编号: 14009 (0x36B9)
- 名称: robot\_config\_clearallerrors\_res
- 描述: 清除机器人当前所有报错的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 36 B9 0F A9 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
