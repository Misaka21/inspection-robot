**电机清错**

**请求**

- 编号: 4151 (0x1037)
- 名称: robot\_config\_motor\_clear\_fault\_req
- 描述: 电机清错
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|motor\_name|string|电机名称（机器人模型中）|否|

**请求示例**


```JSON
{"motor\_name":"motor1"}
// 清错电机名为motor1
5A 01 00 01 00 00 00 17 10 37 00 00 00 00 00 00
7B 22 6D 6F 74 6F 72 5F 6E 61 6D 65 22 3A 22 6D 
6F 74 6F 72 31 22 7D```

**响应**

- 编号: 14151 (0x36E3)
- 名称: robot\_config\_motor\_clear\_fault\_res
- 描述: 电机清错的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 37 47 10 37 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
