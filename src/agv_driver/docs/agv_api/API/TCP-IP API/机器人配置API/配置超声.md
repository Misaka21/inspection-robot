**配置超声**

**请求**

- 编号: 4130 (0x1022)
- 名称: robot\_config\_ultrasonic\_req
- 描述: 配置超声
- JSON 数据区: 见下表


```<p>注意:</p><p>用于配置某个 ID 的超声是否启用</p>```


```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|id|number|超声的 ID|否|
|valid|boolean|是否启动, false = 禁用, true = 启用|否|

**请求示例**


```JSON
5A 01 00 01 00 00 00 16 10 22 00 00 00 00 00 00
{"id":0,"valid":false}
// 序列化后十六进制为：
7B 22 69 64 22 3A 30 2C 22 76 61 6C 69 64 22 3A 
66 61 6C 73 65 7D```

**响应**

- 编号: 14130 (0x3732)
- 名称: robot\_config\_ultrasonic\_res
- 描述: 配置超声的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 37 32 10 22 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
