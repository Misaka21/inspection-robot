**临时修改机器人参数**


```<p>注意：</p><p>请参考 [查询机器人参数 ](..\机器人状态API\查询机器人参数.md)，来设置参数</p>```

**请求**

- 编号: 4100 (0x1004)
- 名称: robot\_config\_setparams\_req
- 描述: 临时修改机器人参数
- JSON 数据区: 见下表


```<p>注意：</p><p>请求数据为 JSON object，其中每一个 key 为不同的插件名，value 为 JSON object，代表当前插件中需要修改的参数</p>```

**修改插件 NetProtocol 和插件 MoveFactory 中相关参数**

**请求示例**


```JSON
{
  "NetProtocol": {
    "RobotName": "RD-TEST-1",
    // 机器人名
    "RobotNote": "test car1" // 机器人备注
  },
  "MoveFactory": {
    "3DCameraHole": true // 导航是否使用3D相机检测坑洞
  }
}```

**响应**

- 编号: 14100 (0x3714)
- 名称: robot\_config\_setparams\_res
- 描述: 临时修改机器人参数的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "create\_on": "2024-05-07T13:54:38.796+0800",
  "ret\_code": 0
}```
