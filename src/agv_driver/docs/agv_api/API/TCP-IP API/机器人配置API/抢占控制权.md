**抢占控制权**

```<p>注意：</p><p>抢占控制权。</p>```

**请求**

- 编号: 4005 (0x0FA5)
- 名称: robot\_config\_lock\_req
- 描述: 抢占控制权
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|nick\_name|string|控制权抢占者名称|否|

**请求示例**


```JSON
{"nick\_name":"srd-seer-mizhan"}
// 控制权抢占者名为srd-seer-mizhan
5A 01 00 01 00 00 00 1F 0F A5 00 00 00 00 00 00
7B 22 6E 69 63 6B 5F 6E 61 6D 65 22 3A 22 73 72 
64 2D 73 65 65 72 2D 6D 69 7A 68 61 6E 22 7D```

**响应**

- 编号: 14005 (0x36B5)
- 名称: robot\_config\_lock\_res
- 描述: 抢占控制权的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 36 B5 0F A5 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
