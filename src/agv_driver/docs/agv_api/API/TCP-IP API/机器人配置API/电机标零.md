**电机标零**

**请求**

- 编号: 4150 (0x1036)
- 名称: robot\_config\_motor\_calib\_req
- 描述: 电机标零
- JSON 数据区: 见下表


```<p>警告：</p><p>目前仅支持 spin 电机，其他类型的电机无视该指令。</p>```


```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|motor\_name|string|电机名称（机器人模型中）|否|

**请求示例**


```JSON
{"motor\_name":"motor1"}
// 标零电机名为motor1
5A 01 00 01 00 00 00 17 10 36 00 00 00 00 00 00
7B 22 6D 6F 74 6F 72 5F 6E 61 6D 65 22 3A 22 6D 
6F 74 6F 72 31 22 7D```

**响应**

- 编号: 14150 (0x36E2)
- 名称: robot\_config\_motor\_calib\_res
- 描述: 电机标零的响应
- JSON 数据区: 见下表

```**字段名**|**类型**|**描述**|**可缺省**``` :- | :- | :- |
|ret\_code|number|API 错误码|是|
|create\_on|string|API 上传时间戳|是|
|err\_msg|string|错误信息|是|

**响应示例**


```JSON
{
  "ret\_code": 0
}```


```Plaintext
5A 01 00 01 00 00 00 0E 37 46 10 36 00 00 00 00
7B 22 72 65 74 5F 63 6F 64 65 22 3A 30 7D```
