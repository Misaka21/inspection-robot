**TCP client 模式**

（3.4.6.13+可用）

模型文件中TCPClient包含五个设置项：ip、端口、超时时间、是否重连、重连次数，超时时间为0时不对超时时间进行设置。

在模型文件中进行设置并保存上传后，Robokit 会尝试连接设置中的ip和端口，连接成功后主动发送一个报文，报文为json，字段如下，服务端需要响应一个符合rbk报文格式的报文，响应报文不做处理，仅代表收到。连接成功后服务端可主动发送请求API。

```字段名|类型|描述``` :- | :- |
|name|string|机器人名称|
|ip|string|客户端ip|
|port|number|客户端端口|
**请求**

同 server 模式

**响应**

同 server 模式
