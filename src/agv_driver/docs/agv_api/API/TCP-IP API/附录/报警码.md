**报警码**

**报警码格式**

```字段名|类型|描述|可缺省``` :- | :- | :- |
||number|字段名为报警码编号，值为时间戳|否|
|desc|string|报警码信息|否|
|describe|string|报警码描述|否|
|method|string|报警处理方法|否|
|reason|string|报警原因|否|
|times|number|报警次数|否|
|dataTime|number|报警发生时间|否（3.4.6.14.20240323版本增加）|
|code|number|报警码编号|否（3.4.6.14.20240323以后版本增加）|
|<p>timestamp</p><p></p>|number|报警时间戳|否（3.4.6.14.20240323以后版本增加）|
**示例**

```JSON
{
  "52101": 1712458103,
  "code": 52101,
  "dataTime": "2024-04-07T10:48:23.691+0800",
  "desc": "laser data invalid",
  "describe": "laser data invalid",
  "method": "",
  "reason": "",
  "times": 1,
  "timestamp": 1712458103
}```
